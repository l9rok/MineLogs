Блоки боксы цвета

Title

Свойство CSS border определяет рамку для всех четырёх сторон элемента. Также можно задать свойство для конкретной стороны: border-top, border-right, border-bottom и border-left. Стандартно в значения входит: толщина стиль цвет. Но также каждое из этих значений можно определить своим свойством border-width, border-style, и border-color. Толщина указывается в px или других единицах измерения. Стиль может быть solid (сплошная), dotted (пунктирная), dashed (пунктирная длинная), double и другие. Форматы цвета уже объяснялись ранее.
